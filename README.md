# React Query Pagination (TypeScript)

A production-ready React + TypeScript project demonstrating:
- React Query pagination
- Query caching & staleTime optimization
- Prefetching next page
- Skeleton loaders
- Memoized list items to avoid re-renders

## Tech Stack
- React + TypeScript
- @tanstack/react-query
- Axios
- Vite
